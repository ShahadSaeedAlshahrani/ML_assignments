# Lab4ML
In this lab I learned how to clean and prepare data for machine learning
I worked with a real chocolate sales dataset 
I fixed wrong data types, handled missing values using median and removed outliers using the IQR method 
I also normalized the data using Min Max and Z-score so features are on the same scale 
For PCA I checked the correlation first and since it was close to zero I concluded that PCA wouldn't be useful in this case 
Overall I realized that preprocessing is just as important as the model itself.
